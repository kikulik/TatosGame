GAME DESIGN & DEVELOPMENT
-------------------------
Created with HTML5 Canvas and Vanilla JavaScript
No external libraries or frameworks used

GRAPHICS
--------
All graphics are procedurally generated using the HTML5 Canvas 2D API.
No external sprite assets are used in this version.

Zombie and boss designs are created using:
- Canvas primitive shapes (circles, rectangles, paths)
- Dynamic color schemes
- Real-time animation calculations

AUDIO
-----
All sound effects are procedurally generated using the Web Audio API.
No external audio files are used.

Sound generation techniques include:
- Oscillator-based tone generation
- Noise buffers for explosion effects
- Frequency modulation for dynamic sounds
- Envelope generators for natural decay

FONTS
-----
System fonts used:
- Segoe UI (Windows)
- Tahoma
- Geneva
- Verdana
- Arial (fallback)
